# Data-Classification-Using-AI
# 🌸 Data Classification Using AI

## Project 2 - DecodeLabs AI Internship

This project demonstrates a Machine Learning classification model using the Iris dataset and a Decision Tree Classifier from Scikit-learn.

## 📌 Project Overview

The model is trained on the famous Iris dataset to classify flowers into one of three species:

- Setosa
- Versicolor
- Virginica

The project also allows users to enter custom flower measurements for real-time prediction.

---

## 📂 Dataset

- Dataset: Iris Dataset
- Total Samples: 150
- Features: 4
- Classes: 3

Features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
